Role Overview:

As Fullstack Developer I participed in a system for register the projects, skills, skill level, status, occupation time, level in each skill, areas, vacations and assignments of each employee.

The challenges in which I support:

  • Implementation of a new dynamic call to assign a call to employees once a month for interaction between them.
  • Implementation of a new section in the application to generate dynamic reports by date to obtain a summary of each employee
  • Implementation of a new section to assign the corresponding area to each employee
  • Fixing logic bugs
  • Migrating API’s from REST to Graphql
  • Implementing a new tool to identify an employee using the phone camera
